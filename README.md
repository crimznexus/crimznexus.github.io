## 💻 Local Development

To run this portfolio locally, follow these steps:

1. **Clone the repo:**
   ```bash
   git clone https://github.com/crimznexus/mustafa.github.io.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm run dev
   ```
